En esta web se tratan imágenes Dicom partiendo de la base que proporciona las librerías CornerStone y DicomParse. Desde esta web es posible filtrar sobre los datos en base al identificador del paciente y el número de cortes que se deseen filtrar.

Cuando se muestra la pantalla de la serie se muestran con ellas los datos del paciente. También es posible cambiar el brillo/contraste de la series que se visualiza.
